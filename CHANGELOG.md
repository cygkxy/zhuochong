# 更新日志

## V1.2

**发布日期：** 2026-05-09

### 介绍
纯白主题切换，全面简化 API 配置，优化主动说话和气泡交互体验。

### 界面变更
- 🎨 **纯白背景 + 黑色字体** — 整体主题从暗色切换为亮色
- 🪟 **对话框简化** — 去掉标题栏和白色背景，只保留输入区 + 发送按钮
- 🫧 **气泡自适应宽度** — 根据文字长度自动调整宽窄
- 🎞️ **切换间隔增加「不切换」** — 选后动画定格在当前画面
- 📌 **移除固定画面/跳舞功能** — 精简交互选项

### API 变更
- 🔧 **只保留 DeepSeek** — 移除 OpenAI / Claude / 自定义等 Provider 选择
- 🐛 **修复「出错了：None」** — PEP 3110 except 变量提前清除问题
- 🐛 **HTTP 错误详情显示** — 显示 API 返回的具体错误信息
- 🤖 **主动说话优化** — 时段感知、多样 prompt、上下文连贯
- 🐛 **主动说话重复切换修复** — 移除冗余的 `switch_gif` 调用
- 🔇 **主动说话静默失败** — API 异常不再弹错误气泡

### 功能
- 💬 **自动根据对话文字数量调整气泡大小**
- 📌 **对话框增加关闭按钮** — 右上角 ✕
- 🗑️ **移除内置固定台词** — 宠物只通过 AI 聊天说话
- 🗑️ **移除右键菜单跳舞和固定画面项**

### 安装包
- `dist/Rainbow桌宠.exe` — Windows 可执行文件（29MB）

## V1.1

**发布日期：** 2026-05-08

### 介绍
对核心代码进行了深度重构，消除大量重复逻辑，同时全面升级 UI 视觉风格——采用赛博科技暗色主题，提供更精致的交互体验。

### 代码重构
- ♻️ **统一 API 请求层** — 将 8 个重复的 API 调用方法合并为 1 个 `_api_request`，消除约 200 行冗余代码，同时修复了 2 个从未被调用的死方法
- 🧹 **移除死代码** — 删除了 `_on_mouse_move` 废弃方法及其绑定
- 📦 **新增依赖声明** — 添加 `requirements.txt`，新用户无需猜测依赖
- 🐛 **修复启动脚本** — `启动桌宠.bat` 路径硬编码修复为 `%~dp0` 相对路径

### UI 升级
- 🎨 **统一主题色系统** — 所有颜色收敛到 `THEME` 字典，全局一致性管理
- 🌃 **赛博科技暗色主题** — 深空蓝黑底色 + 霓虹青 `#22d3ee` 强调色
- 🪟 **卡片式窗口边框** — 设置、聊天、菜单统一外框分层结构
- 🔧 **设置面板重写** — 卡片式行布局、区段标题左侧色条、输入框边框包裹
- 💬 **聊天气泡美化** — 气泡内容包裹精致边框
- 💬 **聊天窗口升级** — 标题栏加强调色底线、输入框边框包裹、发送按钮主题色

### 安装包
- `dist/Rainbow桌宠.exe` — Windows 可执行文件（28MB）
