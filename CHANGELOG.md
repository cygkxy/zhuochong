# 更新日志

## V1.0 — Rainbow 桌宠 初始版本

**发布日期：** 2026-05-08

### 介绍
Rainbow 桌宠的第一个正式版本，基于 Python/Tkinter 构建的 Windows 桌面宠物应用。支持 24 组 GIF 动画、AI 对话、自动行走、聊天气泡等核心功能，同时提供网页版体验。

### 功能
- 🐾 **桌面宠物展示** — 24 组 GIF 动画循环播放，支持拖拽移动
- 🚶 **自动行走** — 宠物在桌面上随机走动，可配置速度和开关
- 🤖 **AI 对话** — 支持 OpenAI / Claude / DeepSeek 等多种 API 接入
- 💬 **主动聊天** — 定时主动说话，每次内容由 AI 实时生成
- ⌨️ **手动聊天** — 按快捷键打开聊天框，支持联网搜索增强回复
- 🫧 **聊天气泡** — 对话内容以气泡形式显示在宠物头顶
- ⚙️ **配置面板** — 图形化界面调整大小、速度、API 设置、气泡颜色等
- 🌐 **网页版** — `index.html` 提供浏览器端桌宠体验

### 安装包
- `dist/Rainbow桌宠.exe` — Windows 可执行文件（18MB）
